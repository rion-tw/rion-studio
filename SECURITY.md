# Security Policy

Do not disclose suspected vulnerabilities, credentials, tokens, or private
browser data in a public Issue.

Report security concerns privately through
[GitHub Security Advisories](https://github.com/rion-tw/rion-studio/security/advisories/new).
Include the affected Rion Studio version, operating system, impact, and minimal
reproduction details. Please allow reasonable time for investigation before any
public disclosure.

Only the latest published release is actively supported with security fixes.
