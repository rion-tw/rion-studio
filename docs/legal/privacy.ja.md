# Rion Studio プライバシー通知

バージョンおよび発効日：2026-07-14

## 1. 範囲と提供者

本通知は Rion Studio project が提供するデスクトップ App のデータ処理を説明します。個別のサポートまたは連絡窓口はありません。プロジェクト運営のクラウドアカウント、広告、分析テレメトリサービスはありません。

## 2. 端末に保存するデータ

ロール名、起動 URL、メモ、カバー画像、ウィンドウサイズ、ワークスペース、マクロ、設定、ブラウザー設定、状態時刻を OS のアプリデータ領域へ保存します。ロール別プロファイルには Cookie、キャッシュ、履歴、Local／Session Storage、IndexedDB、認証トークンが含まれる場合があります。

パスワードの提供を意図的に求めず、独自のパスワードデータベースも作りません。第三者ページへ入力したパスワードはそのページとブラウザーコンテキストが処理しますが、保存セッション自体は機密データです。

## 3. 処理目的

ローカルデータは、利用者が設定したロール、セッション、ウィンドウ、ワークスペース、設定、マクロの復元に使います。認証状態の証拠は端末内で確認します。App を使うだけで本プロジェクトがローカルデータを受け取ることはありません。

## 4. ネットワーク通信と第三者

- 対象サイトと認証事業者は通常のブラウザー要求を受け取り、各ポリシーで処理します。
- パッケージ版は GitHub Releases へ更新確認し、GitHub は IP、App バージョン、プラットフォーム、通常の要求メタデータを受け取る場合があります。
- CDN 互換 Auto は Google がホストするリソースへ短い到達確認を行い、接続失敗またはタイムアウト時に対応リソースを第三者ミラーへ変更します。ミラーは IP、要求 URL、ヘッダー等を受け取る場合があります。
- カスタムプロキシは経由する通信を処理できます。Rion Studio は利用者が選んだプロキシを検証または管理しません。

CDN 互換は限定的なリソース置換であり、VPN や一般的なプライバシーサービスではありません。

## 5. エクスポートとインポート

JSON エクスポートにはロール名、URL、メモ、カバー画像、配置、マクロ、設定が含まれますが、ブラウザープロファイル、Cookie、ログインセッションは含みません。ファイルは暗号化されず、読める人は内容を確認できます。

## 6. 保存、削除、管理

削除、ロール削除、App データ消去、アンインストール後の user-data 削除まで端末に残ります。ロール削除は metadata とブラウザーディレクトリを削除します。エクスポートは保存先から別途削除してください。同意記録は端末内だけにあり、エクスポートしません。

本プロジェクトはローカル記録を受け取らないため、閲覧、修正、出力、削除は App と OS のファイル管理で行います。適用法上の強行的な権利は制限しません。

## 7. セキュリティと限界

ロール分離と Electron の context-isolated preload bridge を使用しますが、絶対的な安全は保証できません。OS アカウント、ディスク、バックアップ、出力、プロキシ、端末を保護してください。

## 8. 子どもと変更

本プロジェクトが子どもの情報を収集する目的の製品ではありません。必要な場合は保護者の同意を得てください。重要な変更時はバージョンを更新し、App 内で再確認を求めます。
