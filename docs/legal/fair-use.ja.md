# Rion Studio 公正利用規則

バージョンおよび発効日：2026-07-14

## 基本原則

Rion Studio は個人による、人が継続して監督する支援用途を目的とします。使用中はその場にいて、App の操作を理解し、いつでも停止できる必要があります。機能が存在しても、ゲームやサービスが許可しているとは限りません。

## 守ること

- 対象サービスの規約、ゲーム規則、自動化・アンチチート方針、コミュニティ規則、アカウント方針に従うこと。
- マクロは小規模な支援操作に限定し、対象ウィンドウを継続監督すること。
- 予期しない動作、他プレイヤーへの影響、規則違反の危険があれば直ちに停止すること。
- アカウント、セッション、端末、プロキシ、出力データを保護すること。

## 禁止事項

- 不在中の無人 bot、ファーミング、継続プレイ、アカウント操作。
- アンチチート、bot 検出、アクセス制御、速度制限、地域制限、執行システムの回避、無効化、調査。
- バグ悪用、資産複製、ゲーム経済の操作、なりすまし、不公正な競争優位の取得。
- 嫌がらせ、サービス妨害、過負荷、無許可スクレイピング、無権限のアカウントまたはデータアクセス。
- 違法行為、または禁止行為を主目的とする設定の配布。

## 執行と責任

第三者は独自に違反を判断し、アカウントを制限できます。本プロジェクトはプレイを監視せず、個別設定を承認せず、監督利用が許可されることも保証しません。重大な違反があれば利用ライセンスは終了します。
