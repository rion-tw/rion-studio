# Rion Studio 利用規約

バージョンおよび発効日：2026-07-14

## 1. 提供者と同意

Rion Studio は Rion Studio project が提供します。個別のサポートまたは連絡窓口は提供していません。同意欄を選択して続行すると、本規約と公正利用規則に同意し、プライバシー通知を確認したものとみなされます。同意しない場合は使用しないでください。

## 2. 製品の性質

Rion Studio は、分離されたブラウザーロール、ウィンドウ配置、ログインセッション、人が監督するマクロを管理する汎用デスクトップ支援ツールです。本プロジェクトは独立しており、ゲーム、ウェブサイト、認証事業者、Google、Chrome、Facebook その他の第三者とは提携、承認、代理または運営関係にありません。

## 3. ライセンスと制限

合法的な個人利用のため、配布されたバイナリをインストールして使用する、取消可能、非独占、譲渡不可の限定ライセンスを付与します。ソースコード、名称、ロゴ、第三者素材のライセンスは含みません。法律、他者の権利、第三者サービスの規則または公正利用規則に反する使用は禁止します。

## 4. 第三者サービスとアカウント

第三者ページとログインは各提供者が管理します。利用規約、ゲーム規則、自動化・アンチチート方針、コミュニティ規則、アカウント方針を確認する責任は利用者にあります。第三者が本ツールを許可すること、ブラウザーフローを維持すること、アカウントを制限しないことは保証できません。

## 5. ログインセッションと端末の安全

分離ブラウザープロファイルには Cookie、ブラウザーストレージ、履歴、認証トークンが含まれる場合があります。パスワードの提供や保存を意図的に求めませんが、保存セッションは機密性があります。端末アクセス、OS の安全、バックアップ、エクスポートファイル、端末共有前のロール削除は利用者の責任です。

## 6. ネットワーク機能

更新確認は GitHub Releases と通信します。カスタムプロキシは閲覧通信を処理できます。CDN 互換モードは Google 接続を確認し、対応リソースを第三者ミラーへ変更する場合があります。これらは本プロジェクトの管理外です。CDN 互換モードは VPN ではなく、可用性、プライバシー、完全性、地域アクセスを保証しません。

## 7. 可用性と変更

Rion Studio はベータまたはプレリリースの場合があります。機能、互換性、保存形式、配布、可用性は変更または終了することがあります。重要な規約または通知の変更時は再同意を求めます。
