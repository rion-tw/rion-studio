# Rion Studio 公平使用規範

版本及生效日：2026-07-14

## 核心原則

Rion Studio 僅供個人且由真人持續監督的輔助用途。使用期間你必須在場、了解 App 正在執行的操作，並能隨時停止。App 提供某項功能，不代表遊戲或服務允許使用該功能。

## 必須遵守

- 查閱並遵守每個目標服務的條款、遊戲規則、自動化政策、反作弊政策、社群規範及帳號規則。
- 巨集僅用於小型輔助操作，且執行期間必須持續監督受影響的遊戲視窗。
- 巨集或工作階段出現非預期行為、影響其他玩家或可能違規時，立即停止。
- 自行保護帳號、登入工作階段、裝置、Proxy 及匯出資料。

## 禁止行為

- 在本人離開時進行無人 bot、自動農資源、持續遊玩或帳號操作。
- 繞過、停用、探測或規避反作弊、bot 偵測、存取控制、速率限制、區域限制或執法系統。
- 利用漏洞、複製資產、操縱遊戲經濟、冒充他人或取得不公平競爭優勢。
- 騷擾使用者、干擾或過載服務、未經許可擷取資料，或未經授權存取帳號或資料。
- 將 Rion Studio 用於違法活動，或散布主要為禁止行為設計的設定。

## 執行與責任

第三方自行判斷活動是否違反規則，並可不受本規範影響而限制帳號。Rion Studio project 不監控遊玩、不核准個別設定，也不保證人工監督的使用必然被允許。重大違反本規範時，使用條款授予的授權即告終止。
