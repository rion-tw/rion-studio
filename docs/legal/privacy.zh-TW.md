# Rion Studio 隱私權聲明

版本及生效日：2026-07-14

## 1. 範圍與提供者

本聲明說明 Rion Studio project 提供的 Rion Studio 桌面 App 如何處理資料。目前不提供個別客服或聯絡管道。Rion Studio 沒有由本專案經營的雲端帳號、廣告或分析遙測服務。

## 2. 儲存在裝置上的資料

App 會在作業系統的應用程式資料目錄保存角色名稱、啟動網址、備註、封面、視窗尺寸、工作區、巨集、偏好設定、瀏覽器設定及狀態時間。每個角色擁有隔離的瀏覽器設定檔，其中可能包含 Cookie、快取、瀏覽紀錄、Local Storage、Session Storage、IndexedDB 及驗證權杖。

Rion Studio 不會刻意要求你提供密碼，也不另建密碼資料庫。輸入第三方頁面的密碼由該頁面及瀏覽器內容處理；但已保存的瀏覽器工作階段仍可能允許帳號存取，應視為敏感資料妥善保護。

## 3. 處理目的

本機資料用於還原你要求的角色、工作階段、瀏覽器視窗、工作區、設定及巨集。登入證據只在本機檢查，以判斷角色是否看似已驗證。僅因使用本 App，Rion Studio project 不會收到這些本機資料。

## 4. 網路通訊與第三方

- 目標網站及身分驗證服務會收到一般瀏覽器請求，並依其自身政策處理資料。
- 封裝版本會向 GitHub Releases 檢查更新；GitHub 可能取得 IP 位址、App 版本、平台資訊及一般請求中繼資料。
- CDN 相容模式的 Auto 選項可能短暫請求 Google 託管資源測試連線；若存取失敗或逾時，會把支援的資源網址改寫到第三方鏡像，鏡像可能取得 IP 位址、請求網址、標頭及其他一般請求中繼資料。
- 你指定的自訂 Proxy 可接收及處理經其轉送的流量；Rion Studio 不驗證也不控制你選擇的 Proxy。

CDN 相容模式只改寫特定資源，不是 VPN 或一般隱私服務。

## 5. 可攜式匯出與匯入

可攜式 JSON 可能包含角色名稱、啟動網址、備註、內嵌封面、視窗配置、巨集及偏好設定，但不包含瀏覽器設定檔、Cookie 或登入工作階段。匯出檔未加密，任何能讀取檔案的人都能看到內容。匯入只會讀取你選擇的檔案並建立新的本機紀錄。

## 6. 保存、刪除與控制

資料會保留在裝置上，直到你刪除資料、刪除角色、清除 App 資料，或解除安裝後移除 user-data 目錄。刪除角色會移除其 metadata 及角色瀏覽器目錄。可攜式匯出檔會留在你儲存或複製的位置，必須另外刪除。法律同意紀錄只保存在裝置上，不會匯出。

由於本專案不會收到 App 的本機紀錄，查閱、更正、匯出及刪除可透過 App 與作業系統檔案控制完成。本聲明不限制適用法律賦予且不得限制的隱私權利。

## 7. 安全與限制

Rion Studio 使用隔離角色目錄及 Electron context-isolated preload bridge，但任何桌面軟體均無法保證絕對安全。請保護作業系統帳號、磁碟、備份、匯出檔、Proxy 及實體裝置，且不要把角色目錄提供給不應使用該登入工作階段的人。

## 8. 未成年人與變更

Rion Studio 並非為本專案蒐集兒童資料而設計；法律要求時，未成年人應取得監護人同意。資料處理方式或本聲明有重大變更時，會更新版本並要求在 App 內重新確認。
