# Rion Studio 公平使用规范

版本及生效日期：2026-07-14

## 核心原则

Rion Studio 仅供个人且由真人持续监督的辅助用途。使用期间你必须在场、了解 App 正在执行的操作，并能随时停止。App 提供某项功能，不代表游戏或服务允许使用该功能。

## 必须遵守

- 查阅并遵守每个目标服务的条款、游戏规则、自动化政策、反作弊政策、社区规范及账号规则。
- 宏仅用于小型辅助操作，执行期间必须持续监督受影响的游戏窗口。
- 宏或会话出现意外行为、影响其他玩家或可能违规时，立即停止。
- 自行保护账号、登录会话、设备、代理及导出数据。

## 禁止行为

- 在本人离开时进行无人机器人、自动刷取、持续游玩或账号操作。
- 绕过、停用、探测或规避反作弊、机器人检测、访问控制、速率限制、区域限制或执法系统。
- 利用漏洞、复制资产、操纵游戏经济、冒充他人或取得不公平竞争优势。
- 骚扰用户、干扰或过载服务、未经许可抓取数据，或未经授权访问账号或数据。
- 将 Rion Studio 用于违法活动，或分发主要为禁止行为设计的配置。

## 执行与责任

第三方自行判断活动是否违反规则，并可不受本规范影响而限制账号。Rion Studio project 不监控游玩、不批准个别配置，也不保证人工监督的使用必然被允许。严重违反本规范时，使用条款授予的许可即告终止。
