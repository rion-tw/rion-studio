# Rion Studio 隐私声明

版本及生效日期：2026-07-14

## 1. 范围与提供者

本声明说明 Rion Studio project 提供的 Rion Studio 桌面 App 如何处理数据。目前不提供个别客服或联系渠道。Rion Studio 没有由本项目运营的云端账号、广告或分析遥测服务。

## 2. 保存在设备上的数据

App 会在操作系统应用数据目录保存角色名称、启动网址、备注、封面、窗口尺寸、工作区、宏、偏好设置、浏览器设置及状态时间。每个角色拥有隔离浏览器配置文件，其中可能包含 Cookie、缓存、浏览记录、Local Storage、Session Storage、IndexedDB 及身份验证令牌。

Rion Studio 不会刻意要求你提供密码，也不另建密码数据库。输入第三方页面的密码由该页面及浏览器上下文处理；但保存的浏览器会话仍可能允许账号访问，应作为敏感数据保护。

## 3. 处理目的

本地数据用于恢复你要求的角色、会话、浏览器窗口、工作区、设置及宏。登录证据只在本地检查，以判断角色是否看似已验证。仅因使用本 App，Rion Studio project 不会收到这些本地数据。

## 4. 网络通信与第三方

- 目标网站及身份验证服务会收到普通浏览器请求，并依其自身政策处理数据。
- 封装版本会向 GitHub Releases 检查更新；GitHub 可能取得 IP 地址、App 版本、平台信息及普通请求元数据。
- CDN 兼容模式的 Auto 选项可能短暂请求 Google 托管资源测试连接；若访问失败或超时，会把支持的资源网址改写到第三方镜像，镜像可能取得 IP 地址、请求网址、标头及其他普通请求元数据。
- 你指定的自定义代理可接收及处理经其转发的流量；Rion Studio 不验证也不控制你选择的代理。

CDN 兼容模式只改写特定资源，不是 VPN 或通用隐私服务。

## 5. 便携式导出与导入

便携式 JSON 可能包含角色名称、启动网址、备注、内嵌封面、窗口布局、宏及偏好设置，但不包含浏览器配置、Cookie 或登录会话。导出文件未加密，任何能读取文件的人都能看到内容。导入只会读取你选择的文件并创建新的本地记录。

## 6. 保存、删除与控制

数据会保留在设备上，直到你删除数据、删除角色、清除 App 数据，或卸载后移除 user-data 目录。删除角色会移除其元数据及角色浏览器目录。便携式导出文件会留在你保存或复制的位置，必须另行删除。法律同意记录只保存在设备上，不会导出。

由于本项目不会收到 App 的本地记录，访问、更正、导出及删除可通过 App 与操作系统文件控制完成。本声明不限制适用法律赋予且不得限制的隐私权利。

## 7. 安全与限制

Rion Studio 使用隔离角色目录及 Electron context-isolated preload bridge，但任何桌面软件都无法保证绝对安全。请保护操作系统账号、磁盘、备份、导出文件、代理及物理设备，不要把角色目录提供给不应使用该登录会话的人。

## 8. 未成年人及变更

Rion Studio 并非为本项目收集儿童数据而设计；法律要求时，未成年人应取得监护人同意。数据处理方式或本声明发生重大变更时，会更新版本并要求在 App 内重新确认。
