# 1.14.0 (2026-07-14)

### Bug Fixes

* **browser:** align Windows Chrome visible frames
* **build:** prevent duplicate Windows helper manifest

### Features

* **browser:** add support for reduced transparency in divider visuals
* **macros:** allow partial role execution
* **settings:** add "Open" label for legal documents in multiple languages
* **settings:** add divider controls and group game options
* **settings:** add workspace appearance controls
* **settings:** separate general and system settings sections with translations
* **settings:** show graphics restart prompt in dialog
* **tests:** update path handling for helper executable in tests
* **workspaces:** add mixed-row layouts
* **workspaces:** improve resize feedback and snapping
* **workspace:** update layout styling for improved responsiveness

### Performance Improvements

* **app:** reduce startup and multi-role overhead
