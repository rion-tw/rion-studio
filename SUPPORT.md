# Support

Use [GitHub Issues](https://github.com/rion-tw/rion-studio/issues) for product bug
reports and feature requests. Include the Rion Studio version, operating system,
expected behavior, actual behavior, and reproducible steps.

Do not post passwords, access tokens, browser profiles, session data, or security
vulnerabilities publicly. Security concerns belong in a private
[security advisory](https://github.com/rion-tw/rion-studio/security/advisories/new).

This public repository does not accept source-code pull requests.
